## 프로젝트 개요 

로봇팔의 원활한 **Pick and Place** 동작 수행을 위해서, Perception 부터 Manipulation 까지 직접 개발하는 것을 목표로 하고 있습니다. <br/>
For the smooth Pick and place operation of the robot arm, we want to develop from Perception to Manipulation.
